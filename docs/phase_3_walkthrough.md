# Walkthrough: Phase 3 - Customer Web Application Complete

I have built a high-end, interactive, and professional landing page for GearFlow using a Black and Neon Green aesthetic.

## 1. Visual Identity
- **Color Palette:** Deep Black background (`#000000`) with high-vibrancy Neon Green accents (`#39FF14`).
- **3D Hero Section:** Features a custom-generated futuristic motorcycle with glowing neon LEDs, using 3D-like perspective and `framer-motion` animations.
- **Glassmorphism:** Used for the navbar and floating cards to create a premium, depth-rich feel.

## 2. Interactive Features
- **Booking Engine:** Integrated a smart `BookingModal` that:
    - Captures customer contact and vehicle details.
    - Features a **Live Location Tagging** system using the Geolocation API.
    - Simulates the hand-off to the backend for the 200 visit fee generation.
- **RAG Chatbot:** Deployed a floating `ChatBot` component:
    - Pre-configured with GearFlow's specific RAG personality.
    - Answers service queries and guides users through the funnel.
    - Features smooth, responsive message bubbles and typing indicators.

## 3. Tech Stack Used
- **Next.js 15 (App Router)**
- **Tailwind CSS v4** (with custom Neon Glow utilities)
- **Framer Motion** (for the "wow" factor animations)
- **Lucide React** (for professional iconography)
