# GearFlow: Fully Autonomous Bike Service

GearFlow is a next-generation, zero-trust autonomous bike service company orchestrated by AI and n8n.

## Core Features
- **Smart Dispatch:** Automated mechanic allocation based on proximity, rating, and availability.
- **AI-Driven Trust:** Vision-based part verification and proximity-locked cancellations.
- **Full Automation:** n8n-driven workflows from lead intake to weekly payouts.
- **Seamless UX:** Premium Next.js customer portal and offline-first mechanic mobile app.

## Tech Stack
- **Frontend:** Next.js (React), TailwindCSS.
- **Mobile:** React Native (Expo).
- **Backend:** Node.js (Express) / PostgreSQL.
- **Orchestration:** n8n.
- **AI:** Groq, OpenAI (Vision), Vector DB (Pinecone/Qdrant).

## Status
Currently in **Phase 2: Backend & Infrastructure**.
